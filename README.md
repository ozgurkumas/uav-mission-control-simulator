\# UAV Mission Control \& Telemetry Simulator



A Java-based simulator for managing UAV missions, flight states, route planning, telemetry processing, and system alerts.



\## Project Goals



This project is being developed to practise and demonstrate:



\* Object-oriented programming

\* Java Collections Framework

\* Data structures and graph algorithms

\* Design patterns

\* Unit testing

\* SQL and JDBC

\* Concurrent programming

\* Git and GitHub workflows



\## Planned Features



\* UAV registration and management

\* Mission and waypoint creation

\* Flight state management

\* Route planning using graph algorithms

\* Telemetry generation and processing

\* Alert generation

\* Persistent storage using JDBC

\* Basic JavaFX user interface



\## Technologies



\* Java 21

\* Maven

\* JUnit

\* Git and GitHub



\## Project Status



The project is currently in its initial setup and domain modelling phase.
